Data Types in Java Sript :

         I) PRIMITIVE :

                                      1) STRING :
                                                                     represents sequence of characters e.g. "hello"

                                      2)NUMBER :
  			        	represents numeric values e.g. 100
                                      3)BOOLEAN :
                                                                     represents boolean value either false or true
                                      4)UNDEFINED :
			       	represents undefined value
                                      5)NULL :
   
                                                                	represents null i.e. no value at all .
      II) NON-PRIMITIVE :
  

                                      1)OBJECT :	represents instance through which we can access members
                                      
                                     2) ARRAY  :              represents group of similar values

                                    3) RegExp    :	represents regular expression
